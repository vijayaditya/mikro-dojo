# MIKRO-DOJO Project Identity & Branding Guidelines

## Project Name

**MIKRO-DOJO** (Multi-view Iterative Kinematic Reinforcement Optimization - Demonstration Observation Joint Optimization)

### Name Rationale
- **MIKRO**: Emphasizes the micro/small scale of RC vehicles while creating a unique, trademarkable identity
- **DOJO**: Conveys the concept of a training ground where mastery is achieved through repeated practice
- **Combined**: Creates a memorable brand that balances technical sophistication with approachability

## Core Identity

### Mission Statement
MIKRO-DOJO is a reality-native robot learning system that enables RC-scale vehicles to acquire complex motor skills through safe iteration, multi-view observation, and intelligent evaluation—achieving 1000x more learning cycles than traditional approaches.

### Vision
To democratize robot learning research by providing an accessible, affordable, and scientifically rigorous platform that bridges the gap between simulation and reality.

### Core Values
- **Iteration Over Perfection**: Embrace failure as the primary teacher
- **Reality Over Simulation**: Learn in the physical world first
- **Observation Over Assumption**: Use multiple perspectives to understand behavior
- **Open Over Closed**: Share knowledge to advance the field

### Brand Terminology

**System Components**:
- **Mikro-Dojo Arena**: The physical bounded learning environment
- **MD Protocol**: The standardized learning methodology
- **Dojo Session**: A complete learning cycle for a skill
- **Mikro Trial**: Individual execution attempt

**Skill Hierarchy**:
- **White Belt Skills**: Basic movements (forward, reverse, turns)
- **Yellow Belt Skills**: Intermediate maneuvers (parking, slalom)
- **Green Belt Skills**: Advanced techniques (controlled slides, ramps)
- **Brown Belt Skills**: Complex maneuvers (drifts, donuts, j-turns)
- **Black Belt Skills**: Expert stunts (barrel rolls, wall rides, combinations)

**Technical Terms**:
- **MD-Trained**: Models trained using the MIKRO-DOJO system
- **Dojo Masters**: Successfully learned and deployed skills
- **Sensei Mode**: Human demonstration phase
- **Kata Library**: Collection of learned behavioral primitives

## Visual Identity Guidelines

### Logo Specifications

**Primary Logo Elements**:
- Stylized "MD" monogram with dynamic motion lines
- Optional: Small RC car silhouette integrated into letterforms
- Aspect ratio: 1:1 (square) or 2:1 (horizontal)

**Logo Variations**:
1. **Full Logo**: MIKRO-DOJO with icon
2. **Abbreviated**: MD monogram only
3. **Text Only**: MIKRO-DOJO wordmark

### Color Palette

**Primary Colors**:
- **Mikro Blue**: #0066CC (RGB: 0, 102, 204) - Intelligence, depth, trust
- **Dojo Orange**: #FF6600 (RGB: 255, 102, 0) - Energy, innovation, action

**Secondary Colors**:
